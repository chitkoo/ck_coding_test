## Running the project

To run this project, open your desired emulator/simulator.

```bash
  cd your/directory
  git clone project
```

```bash
  cd project/
  
  flutter pub get

  flutter run
```

For a better performance, run using this

```bash
  flutter run --profile
```

*Note that running profile mode is not supported for IOS simulator.
